Лабораторная работа №1
Умножение квадратных матриц

Задание
Написать программу на C++ для перемножения двух квадратных матриц.
Исходные данные: файлы с матрицами.
Выходные данные: файл с результирующей матрицей, время выполнения и объем вычислений.
Обязательна проверка результатов с использованием Python/NumPy.

Реализация
Программа реализует классический алгоритм умножения матриц с тремя вложенными циклами.
Сложность алгоритма: O(N³).
Объем вычислений: 2 × N³ операций.
Время выполнения измеряется с помощью <chrono>.

Верификация
Для проверки корректности использован Python + NumPy.
Во всех тестах результаты совпали (C++ = NumPy % 256).

Исследование зависимости времени от размера матрицы
Размер N	Время (мс)
50	        0.130
100	        0.460
200	        2.742
400	        23.254
Анализ: При увеличении размера матрицы в 2 раза время выполнения растет примерно в 8 раз (2³), что соответствует теоретической сложности O(N³).

Вывод
Программа корректно перемножает квадратные матрицы uint8.
Результаты верифицированы с помощью Python/NumPy.
Время выполнения растет пропорционально N³, подтверждая теоретическую оценку сложности алгоритма O(N³).
